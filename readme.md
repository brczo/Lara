Lara是一款基于Laravel框架5.5版本开发的个人博客系统,它灵活轻巧,简单易用并易于扩展,适合搭建个人博客或作为企业CMS使用.

V1.0 版本主要特性:

 1. 文章管理
 2. 分类管理(支持无限级分类)
 3. 注册用户管理
 4. 角色管理
 5. 评论管理
 6. 标签管理
 7. 单页面管理
 8. 链接管理
 9. 集成Quill编辑器,支持图片上传,界面简洁美观
